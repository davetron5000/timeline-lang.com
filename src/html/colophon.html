Timeline is (obviously, I hope) not real. Time travel doesn't exist (that we know of). This site should be thought of as an art project.

The site is built using a combination of Webpack, Handlebars, HTML, Tachyons, and custom JavaScript code. It is all open source. Headers are set in Futura, with body text in Palatino.
